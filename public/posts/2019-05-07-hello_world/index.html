R Markdown

This is my first post! This website was developed with R Markdown, Blogdown, Hugo, and served via Netlify. I tried hosting this on GitHub Pages, but had difficulty with Jekyll and the R Markdown format. Having read about Blogdown, I’m making the transition.

So far this is great. I can embed R code like this:

summary(cars)
##      speed           dist       
##  Min.   : 4.0   Min.   :  2.00  
##  1st Qu.:12.0   1st Qu.: 26.00  
##  Median :15.0   Median : 36.00  
##  Mean   :15.4   Mean   : 42.98  
##  3rd Qu.:19.0   3rd Qu.: 56.00  
##  Max.   :25.0   Max.   :120.00

Or include plots:

pie(
  c(280, 60, 20),
  c('Sky', 'Sunny side of pyramid', 'Shady side of pyramid'),
  col = c('#0292D8', '#F7EA39', '#C4B632'),
  init.angle = -50, border = NA
)
